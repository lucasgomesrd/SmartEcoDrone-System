# Drone Simulation

## Descrição
Módulo responsável por simular a coleta de dados ambientais por um drone autônomo.

## Funcionalidades
- Simulação de temperatura
- Simulação de umidade
- Simulação de coordenadas GPS
- Geração de dados em tempo real

## Objetivo
Representar o comportamento de um drone em campo sem necessidade de hardware físico.

## Tecnologias
- Python

## Como executar
```bash
python simulation.py
